Volleyball Stats Tracker

An Interactive Excel Team Performance Tracker

By Triet Nguyen

A comprehensive volleyball team stats tracker built in Excel that allows users to log every player's performance per set and automatically generates a summary dashboard with key metrics and visualizations. Built for real game use — just enter the numbers and everything updates automatically.

Features

Stats Input Sheet
- Tracks up to 15 players per set
- Records all major volleyball stats per player per set
- Auto-calculates Hit Efficiency, Pass Percentage, and Serve Efficiency
- Color coded player rows for easy identification
- Instructions included so anyone can use it

Summary Sheet
- Auto-generates player totals and averages from the Stats Input sheet
- Kills by Player bar chart
- Hit Efficiency by Player bar chart
- Updates automatically when new data is entered

How to Use
- Download the Volleyball_Stats_Sheet.xlsx file
- Open it in Microsoft Excel
- Go to the Stats Input sheet
- Replace P1-P15 with your actual player names
- After each set enter each player's stats in their row
- Go to the Summary sheet to see totals, averages, and charts update automatically
- For a new match, either make a new sheet with the same template or add new rows below with the new match and set numbers

Efficiency Formulas
- Hit Efficiency = (Kills - Errors) / Attempts
- Serve Efficiency = (Serves - Errors) / Serves
- Pass Percentage = Perfect Passes / Pass Attempts
